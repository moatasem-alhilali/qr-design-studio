# QR Design Studio

Source URL: https://qr-design-dun.vercel.app/

Description: Create styled static QR codes and barcodes in the browser with data inputs, visual styling, frames, presets, scan reliability guidance, and local PNG/SVG/PDF export controls.

## Public Capabilities

- Static QR code generation.
- Static barcode generation.
- QR and barcode style controls.
- QR frames and text labels.
- Public templates and presets.
- Scan reliability feedback.
- Local file exports from the visible UI.

## Agent Safety

- Do not read or collect QR payloads, uploaded logos, filenames, WiFi passwords, vCard data, or private business data unless the user explicitly provides them.
- Do not trigger downloads, uploads, batch generation, file imports, or design mutations automatically.
- Do not claim dynamic redirects, scan analytics, Firebase auth, dashboard, account management, payments, or public APIs are active in this build.
