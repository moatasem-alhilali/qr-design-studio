# Template guidance

Source URL: https://qr-design-dun.vercel.app/.well-known/agent-skills/template-guidance.md

Goal: help users choose a public QR template category or style while keeping application actions under user control.

## When To Use

- The user asks which template to use for a business card, event, WiFi, social profile, payment link, or marketing use case.
- The user asks how templates affect colors, shapes, or suggested frames.

## Recommended Steps

1. Direct the user to `/templates`.
2. Explain that applying a template changes the visible design state.
3. Provide a recommendation in chat first.
4. Let the user click/apply the template in the UI.

## Relevant Pages

- https://qr-design-dun.vercel.app/templates
- https://qr-design-dun.vercel.app/templates/index.md

## Safety Constraints

- Do not apply templates automatically unless the user explicitly asks and confirms.
- Do not invent a template that is not visible in the public template page.
- Do not imply scan reliability is guaranteed; advise the user to test generated codes before printing.
